# شيفرة كود - موقع Front-End

هذا المشروع هو نسخة Front-End لموقع شركة "شيفرة كود" لتقديم الدورات التدريبية والخدمات، مبني باستخدام **HTML, CSS, JavaScript**.  

---

## الصفحات

1. **الرئيسية (index.html)**  
   - صورة Hero بسيطة.  
   - نص تعريفي عن الشركة.  
   - زر احجز دورة يفتح صفحة الدورات.  

2. **الدورات (courses.html)**  
   - عرض الدورات باستخدام Cards.  
   - كل دورة تحتوي: الاسم، وصف مختصر، المدة، السعر.  
   - زر "احجز الآن" يفتح نموذج الحجز.  

3. **نظام الحجز**  
   - نموذج حجز يحتوي: الاسم، رقم الجوال، البريد، اختيار الدورة.  
   - بعد الإرسال يظهر تنبيه نجاح.  
   - البيانات تُحفظ في `localStorage`.  

4. **أعمالنا / Portfolio (portfolio.html)**  
   - شبكة صور من صفحة الإنستغرام.  
   - عند الضغط على الصورة، تظهر أكبر داخل Modal.  

5. **تواصل معنا (contact.html)**  
   - نموذج لإرسال رسالة.  
   - روابط السوشال ميديا (Instagram موجودة).  

---

## المميزات

- موقع **Responsive** يعمل على جميع الأجهزة.  
- تصميم مرتب وواضح.  
- نظام الحجز فعال ويخزن البيانات محلياً.  
- Modal لعرض صور Portfolio.  

---

## كيفية التشغيل

1. قم بتحميل المشروع كامل أو استنساخه من GitHub.  
2. تأكد أن المجلد `images/` يحتوي على الصور التالية:  
   - IMG_4992.jpeg  
   - IMG_4993.jpeg  
   - IMG_4994.jpeg  
   - IMG_4995.jpeg  
3. افتح أي صفحة HTML في المتصفح، مثل: `index.html`  
4. للتأكد من عمل الحجز، افتح صفحة الدورات وجرب نموذج الحجز.  

---

## ملاحظات

- المشروع يستخدم فقط **Front-End**، لا يوجد Back-End.  
- جميع البيانات يتم حفظها مؤقتًا في المتصفح عبر `localStorage`.
